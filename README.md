# Baklawa Bites Website (GitHub-ready)

This is a static website bundle (HTML/CSS/JS + assets) you can push directly to GitHub and deploy with GitHub Pages.

## Files
- `index.html`
- `css/styles.css`
- `js/app.js`
- `assets/images/`

## Run locally
```bash
python3 -m http.server 8080
```
Open http://localhost:8080
